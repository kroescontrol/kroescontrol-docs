# Kroescontrol Documentation 📚

**Nederlands: Deze repository bevat de publieke Kroescontrol documentatiewebsite**

## 🚀 Over Kroescontrol Documentatie

Deze repository bevat de broncode voor de publieke Kroescontrol documentatiewebsite, gebouwd met [Nextra](https://nextra.site/). De website biedt documentatie over:

- Algemene bedrijfsinformatie en cultuur
- Werken bij Kroescontrol  
- Branding en huisstijl
- Kantoor en faciliteiten
- Freelancecontrol model

### Functies

- Modern documentatie framework met Nextra
- Snelle navigatie en zoekfunctionaliteit
- Responsive design voor alle apparaten
- Dark mode ondersteuning

### Ontwikkeling

```bash
# Installeer dependencies
npm install

# Start lokale ontwikkelingsserver
npm run dev

# Bouwen voor productie
npm run build

# Start productie server
npm start
```

### Bezoek de website

De live documentatie is beschikbaar op [docs.kroescontrol.nl](https://docs.kroescontrol.nl)

### Deployment

Deze site wordt automatisch gedeployed naar Vercel bij elke push naar de main branch.
